# TODO Riverpod MVVM (Flutter)

Flutter TODO app in **MVVM** architecture using **Riverpod** and **Cloud Firestore** for real-time sync and sharing.

Features
- Share tasks with other users by email/ID.
- Realtime updates: when a shared user edits a task, owner sees changes instantly via Firestore listeners.
- Share via system share (email, messenger, etc.) using `share_plus`.
- Infinite/paginated list (Firestore query with pagination).
- UI components as reusable widgets.
- Responsive layout.

**Important:**  You need to configure Firebase (Android / iOS / Web) and add platform-specific config files.
